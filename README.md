# Foodie - Ứng dụng chia sẻ công thức nấu ăn

Ứng dụng mobile và web để chia sẻ, khám phá và lưu trữ các công thức nấu ăn Việt Nam.

## 📁 Cấu trúc dự án

```
Foodie/
├── foodie-backend/     # Backend API (Node.js + Express + MongoDB)
├── mobile/            # Mobile App (React Native + Expo)
└── foodie-admin/      # Admin Web (React + Vite)
```

## 🚀 Bắt đầu

### Backend
```bash
cd foodie-backend
npm install
npm start
```

### Mobile App
```bash
cd mobile
npm install
npx expo start
```

### Admin Web
```bash
cd foodie-admin
npm install
npm run dev
```

## 📝 Tính năng chính

- 🔐 Đăng nhập/Đăng ký
- 📖 Xem và tìm kiếm công thức
- 💾 Lưu công thức yêu thích
- 📝 Tạo và chia sẻ công thức
- ⭐ Đánh giá công thức
- 👥 Quản lý người dùng và nội dung (Admin)

## 🛠️ Công nghệ

- **Backend**: Node.js, Express, MongoDB, JWT
- **Mobile**: React Native, Expo, TypeScript
- **Admin**: React, Vite, Tailwind CSS
