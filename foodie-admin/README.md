# Foodie Admin Web

Giao diện quản trị web cho ứng dụng Foodie, xây dựng với React và Vite.

## 🚀 Cài đặt

```bash
npm install
```

## ⚙️ Cấu hình

Tạo file `.env`:

```env
VITE_API_URL=http://localhost:8080/api
```

## 🏃 Chạy ứng dụng

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## 👨‍💼 Tính năng Admin

- 📊 Dashboard thống kê
- 👥 Quản lý người dùng
- 📖 Quản lý công thức
- ✅ Duyệt/từ chối công thức
- 📝 Quản lý bài đăng
- 🗂️ Quản lý danh mục
- 📈 Xem báo cáo

## 🛠️ Công nghệ

- React
- Vite
- Tailwind CSS
- React Router
- Axios
